# **📌 Description**

This repository contains a collection of Microsoft Excel projects focused on data analysis, data cleaning, reporting, and dashboard creation using real-world datasets. The projects demonstrate practical Excel skills used in business and data analysis environments, including data manipulation, formula-based calculations, and visualization techniques.

The repository showcases hands-on experience in organizing and analyzing data, creating dynamic dashboards, and generating meaningful insights to support data-driven decision-making.
